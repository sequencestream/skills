# Skill Improver

Improve your skill. Scores the target across name, description, scope, clarity, examples, safety, and progressive disclosure. Default behavior audits and lists improvements; rewriting only after user approval.

See [`SKILL.md`](./SKILL.md) for the full skill definition.

## Install

Installed via [skilladmin](https://github.com/sequencestream/skilladmin).

```bash
skilladmin install https://github.com/sequencestream/skills/ss-skill-improver
```

## Usage

```
/ss-skill-improver <path-to-SKILL.md>
/ss-skill-improver <skill-name>
```
